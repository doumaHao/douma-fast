
## 动态代理 ##
一共有4中动态代理方式：
- **Java Proxy**
- **Cglib**
- **AspectJ**
- 

### Java Proxy ###
特点：
* 增加代理对象么 -> 是的
* 必须要接口 -> 是的 (代理实现类就是代理对象的实现类)
* 无法代理final方法 -> 是的

### Cglib ###
特点：
* 增加代理对象 -> 是的
* 必须接口 -> 不是 (代理实现类是代理对象的子类)
* 无法代理final方法 -> 是的

### AspectJ ###
特点：
* 增加代理对象 -> 不是（是修改原class二进制文件）
* 必须接口 -> 不是
